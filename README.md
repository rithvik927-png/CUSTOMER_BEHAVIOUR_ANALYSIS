# 🛍️ Customer Behavior Analysis | End-to-End Data Analytics Project

## 📌 Project Overview

This project demonstrates a complete end-to-end data analytics workflow focused on understanding customer purchasing behavior and generating actionable business insights. The objective is to transform raw customer transaction data into meaningful analytics that support data-driven decision-making.

The project covers the full analytics lifecycle, including data preparation, exploratory analysis, SQL-based business problem solving, dashboard development, and business reporting.

---

## 🎯 Business Objectives

The primary goals of this project are:

* Analyze customer purchasing patterns and shopping behavior
* Identify key factors influencing purchase decisions
* Segment customers based on purchasing activity
* Measure the impact of discounts, subscriptions, and seasonal trends
* Create interactive dashboards for business stakeholders
* Generate actionable recommendations to improve customer engagement and sales performance

---

The project follows a modern analytics workflow:

```bash

```

### Step 2: Run Python Analysis

Open:

```text
Customer_Shopping_Behavior_Analysis.ipynb
```

Execute all cells to perform:

* Data Cleaning
* Exploratory Data Analysis
* Data Transformation

### Step 3: Load Data into SQL Database

* Create a database
* Import processed data
* Execute SQL scripts

